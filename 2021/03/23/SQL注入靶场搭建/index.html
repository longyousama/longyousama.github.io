0x00.前言

最近想把Apache+PHP+MySQL，Asp.net+IIS+SQLServer，Jsp+Oracle的搭建注入环境的过程记录一下

0x01.MySQL注入靶场(php5.x版本)

  • 使用集成环境小皮面板和xmapp搭建出Apache+PHP+MySQL，将mysql-sqli-labs解压根目录后修改sqli-labs-master/sql-connections/db-creds.inc文件中的mysql账号和密码

    下载地址：https://github.com/Audi-1/sqli-labs

  • 访问sqli-labs-master靶场创建数据库内容

安装失败

失败原因：PHP版本=>7.x，将php版本修改为5.x版本或下载php7.x注入版本靶场

php7.x注入版本靶场地址：https://github.com/Rinkish/Sqli_Edited_Version

0x02.SQLServer注入靶场(sqlserver2008r2)

SQLServer2008r2下载地址：https://pan.baidu.com/s/1F9003Z6uhlyYeQmKNjz6Vg 提取码：8k22

SQLServer2008r2安装教程：https://jingyan.baidu.com/article/eae078276b7e741fed54857b.html

靶场下载地址：https://github.com/Larryxi/MSSQL-SQLi-Labs

访问http://ip/MSSQL-SQLi-Labs-master/install.asp，在ASP中启动父路径、在错误页启动详细信息、在ASP中设置“将错误发送到浏览器”的值设为 “true”，如果无法创建报错提示没有test数据库，需要用SQL Server management studio创建test数据库。
